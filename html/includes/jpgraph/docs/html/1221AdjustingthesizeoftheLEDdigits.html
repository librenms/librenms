12.2.1 Adjusting the size of the LED digits

There are two parameters that can be used to alter the size of the LED digits.
  • The size (in pixels) of each "led diode"
  • The distance, specified as fractions of the "led diode" size between each "led diode"
These two parameters are given in the constructor which has the signature DigitalLED74::DigitalLED74($aRadius=2,$aMargin=0.6) as can be seen from the above signature the default size is 2 pixels in size and a fraction of 0.6 as the distance.
